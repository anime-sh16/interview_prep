# ML / AI Concepts from Scratch

Implementations of core ML and AI concepts written from scratch in PyTorch, with a focus on deep understanding rather than using high-level abstractions.

Each topic is self-contained and includes:
- **Python implementations** — clean, well-structured PyTorch code
- **Q&A** — interview-level questions and answers
- **Cheat sheet** — concise reference for quick revision
